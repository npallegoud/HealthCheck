# HealthCheck
Health Check of Linux Instance

Download the monitor.sh file

Copy to /usr/bin

Commands:

**sudo cp monitor.sh /usr/bin/monitor**

**sudo chmod 777 /usr/bin/monitor**

From any user we can execute :

Usage:

**monitor**
